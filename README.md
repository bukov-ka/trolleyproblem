# Trolley Problem Game

An interactive browser-based game exploring the famous philosophical thought experiment known as the "Trolley Problem."

## About the Game

The trolley problem is a series of thought experiments in ethics and psychology, involving stylized ethical dilemmas of whether to sacrifice one person to save a larger number. In this game, you control a train switch that determines which track the train will take, and consequently, which people will be affected.

- 8 different decision scenarios
- Real-time train simulation
- Decision tracking and analysis
